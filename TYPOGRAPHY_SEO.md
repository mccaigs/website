```typescript

      color: theme('colors.gray.800'),

        fontWeight: '500',
        '&:hover': {
          textDecoration: 'underline',
          color: '#124E75',  // Darker on hover
        },
      },
      
      // Blockquotes - Brand accent
      blockquote: {
        borderLeftColor: theme('colors.brand'),
        color: theme('colors.gray.600'),
        fontStyle: 'italic',
        backgroundColor: theme('colors.gray.50'),
        borderLeftWidth: '4px',
        borderRadius: '0 0.5rem 0.5rem 0',
      },
      
      // Inline code - Brand colored
      code: {
        backgroundColor: theme('colors.gray.100'),
        color: theme('colors.brand'),
        padding: '0.125rem 0.25rem',
        borderRadius: '0.25rem',
        fontWeight: '500',
      },
      
      // Code blocks
      pre: {
        backgroundColor: theme('colors.gray.900'),
        color: theme('colors.gray.100'),
      },
    },
  },
}
```

## üîç SEO Implementation

### Comprehensive Metadata

**File:** `/app/blog/[slug]/page.tsx`

```typescript
export async function generateMetadata({ params }) {
  const { slug } = await params;
  const post = await getMDXContent(slug);

  return {
    // Page title
    title: `${frontmatter.title} | McCaigs AI Blog`,
    
    // Description
    description: frontmatter.excerpt,
    
    // Keywords
    keywords: frontmatter.tags?.join(', '),
    
    // Authors
    authors: [{ name: frontmatter.author }],
    
    // Open Graph (Facebook, LinkedIn)
    openGraph: {
      title: frontmatter.title,
      description: frontmatter.excerpt,
      type: 'article',
      publishedTime: frontmatter.date,
      authors: [frontmatter.author],
      tags: frontmatter.tags,
      images: [
        {
          url: frontmatter.image,
          width: 1200,
          height: 630,
          alt: frontmatter.title,
        },
      ],
    },
    
    // Twitter Card
    twitter: {
      card: 'summary_large_image',
      title: frontmatter.title,
      description: frontmatter.excerpt,
      images: [frontmatter.image],
      creator: '@McCaigsAI',
    },
  };
}
```

## üìù Blog Post Structure

### Semantic HTML

```tsx
<article className="prose prose-lg dark:prose-invert max-w-none">
  <header className="not-prose">
    {/* Tags */}
    {/* Title (h1) */}
    {/* Date & Author */}
    {/* Featured Image */}
  </header>
  
  {/* MDX Content - Auto-styled by prose */}
  {content}
  
  <footer className="not-prose">
    {/* Publication info */}
    {/* Navigation links */}
  </footer>
</article>
```

### Prose Classes

**Applied:** `prose prose-lg dark:prose-invert max-w-none`

- `prose` - Base typography styles
- `prose-lg` - Larger text (18px base)
- `dark:prose-invert` - Dark mode variant
- `max-w-none` - Remove max-width constraint

### Not-Prose Sections

Elements with `not-prose` class opt out of prose styling:
- Header (custom styling for metadata)
- Footer (custom CTA buttons)

## üîß MDX Frontmatter Requirements

### Required Fields

```yaml
---
title: "Post Title"              # SEO title, H1
date: "YYYY-MM-DD"               # ISO 8601 format
author: "Author Name"             # Byline, author meta
excerpt: "Short description"      # Meta description, OG description
image: "/assets/blog/image.jpg"  # OG image (1200x630px)
tags: ["Tag1", "Tag2"]           # Keywords, categories
---
```

### SEO Best Practices

1. **Title**: 50-60 characters
2. **Excerpt**: 150-160 characters
3. **Image**: 1200x630px (Open Graph standard)
4. **Tags**: 3-5 relevant keywords
5. **Date**: Use ISO format (YYYY-MM-DD)
